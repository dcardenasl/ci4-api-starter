# Internacionalización (i18n)

**Referencia Rápida** - Para detalles completos ver `../ARCHITECTURE.md` sección 12.

## Archivos de Idioma

```
app/Language/
├── en/                    # Inglés
│   ├── Api.php
│   ├── Auth.php
│   ├── Users.php
│   └── ...
└── es/                    # Español
    └── ... (mismos archivos)
```

## Uso

```php
// En código
$message = lang('Auth.loginSuccess');
// Retorna: "Login successful" o "Inicio de sesión exitoso"

// Con parámetros
$message = lang('Validation.min_length', [8]);

// En excepciones
throw new NotFoundException(lang('Users.notFound'));

// En servicios
return ApiResponse::success($data, lang('Users.updatedSuccess'));
```

## Detección de Locale

Orden de prioridad:
1. Parámetro de consulta: `?lang=es`
2. Header `Accept-Language`
3. Predeterminado de `app/Config/App.php`

## Añadir Traducciones

```php
// app/Language/en/Products.php
return [
    'notFound' => 'Product not found',
    'created' => 'Product created successfully',
];

// app/Language/es/Products.php
return [
    'notFound' => 'Producto no encontrado',
    'created' => 'Producto creado exitosamente',
];
```
