# Internationalization (i18n)

**Quick Reference** - For complete details see `../ARCHITECTURE.md` section 12.

## Language Files

```
app/Language/
├── en/                    # English
│   ├── Api.php
│   ├── Auth.php
│   ├── Users.php
│   └── ...
└── es/                    # Spanish
    └── ... (same files)
```

## Usage

```php
// In code
$message = lang('Auth.loginSuccess');  
// Returns: "Login successful" or "Inicio de sesión exitoso"

// With parameters
$message = lang('Validation.min_length', [8]);

// In exceptions
throw new NotFoundException(lang('Users.notFound'));

// In services
return ApiResponse::success($data, lang('Users.updatedSuccess'));
```

## Locale Detection

Priority order:
1. Query parameter: `?lang=es`
2. `Accept-Language` header
3. Default from `app/Config/App.php`

## Adding Translations

```php
// app/Language/en/Products.php
return [
    'notFound' => 'Product not found',
    'created' => 'Product created successfully',
];

// app/Language/es/Products.php
return [
    'notFound' => 'Producto no encontrado',
    'created' => 'Producto creado exitosamente',
];
```
