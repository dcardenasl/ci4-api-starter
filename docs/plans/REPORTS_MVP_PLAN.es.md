# Plan MVP de Reportes

Resumen en español del plan MVP para reportes.

## Objetivo

Definir un alcance mínimo viable para generación de reportes con contratos claros y bajo acoplamiento.

## Criterios de diseño

1. Contratos DTO-first para entradas y salidas.
2. Servicios puros y testeables.
3. Validación y filtros consistentes.
4. Cobertura de pruebas por capa.
